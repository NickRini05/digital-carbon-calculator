# Digital Carbon Footprint Calculator

A web-based tool that estimates the carbon emissions produced from everyday digital activities such as streaming, browsing, emailing, cloud usage, and device manufacturing.

---

## Purpose

Many people are unaware that digital activities contribute to global carbon emissions through data centers, network infrastructure, and device energy use.

This project was created to:
- Make digital carbon emissions more visible and understandable  
- Help users estimate their personal digital footprint  
- Encourage more sustainable technology habits  

---

## How It Works

Users input their digital activity (such as streaming hours, number of emails, device usage, and cloud storage). The calculator then estimates total annual CO₂ emissions in kilograms.

The results are broken down by category and converted into real-world equivalents, such as miles driven or other relatable comparisons, to make the impact easier to understand.

---

## Key Features

- Real-time CO₂ emissions calculation  
- Breakdown of emissions by digital activity  
- Equivalent comparisons (e.g., driving distance)  
- Interactive and user-friendly interface  
- Visual output using charts and summaries
